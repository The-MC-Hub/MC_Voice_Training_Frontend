# THE MC HUB - COLOR SYSTEM

## Core Palette
- **Primary Navy:** `#002366` - Used for primary headers, navigation, and core branding elements.
- **Accent Gold:** `#FDB813` - Used for calls to action, verified badges, and highlights.
- **Deep Slate:** `#1A1A1A` - Primary background color for dark sections/cards.
- **Paper White:** `#FFFFFF` - Primary text color on dark backgrounds, or background for reading sections.

## Functional Colors
- **Success:** `#28A745` - Verified states and successful bookings.
- **Warning:** `#FFC107` - Pending actions or notifications.
- **Danger:** `#DC3545` - Errors or canceled bookings.
- **Info:** `#17A2B8` - Helpful tips and secondary information.

## Typography
- **Headings:** Inter / Roboto (Sans-Serif)
- **Reading Content:** Playfair Display (Serif) - Specifically for MC Scripts.

## Shadows & Elevations
- **Floating Cards:** `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- **Glass Effect:** `backdrop-filter: blur(8px); background: rgba(255, 255, 255, 0.05); border: 1px solid rgba(255, 255, 255, 0.1);`
